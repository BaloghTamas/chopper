Chopper is a Dart HTTP client generator using source_gen and inspired by Retrofit.

## Installation

```yaml
# pubspec.yaml

dependencies:
  chopper: ^2.0.0

dev_dependencies:
  build_runner: ^1.0.0
  chopper_generator: ^2.0.0
```
